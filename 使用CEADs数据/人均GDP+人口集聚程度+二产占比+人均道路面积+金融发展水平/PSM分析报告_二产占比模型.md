# PSM分析报告 - 二产占比模型

**分析日期**: 2025-01-17

## 控制变量组合

本PSM分析使用以下5个控制变量:

1. **ln_pgdp** - 人均GDP (对数)
2. **ln_pop_density** - 人口集聚程度 (对数)
3. **secondary_share** - 二产占比 (水平值)
4. **ln_road_area** - 人均道路面积 (对数)
5. **financial_development** - 金融发展水平 (水平值)

## 数据准备

### 数据集来源
- **主数据集**: `总数据集_2007-2023_完整版_无缺失FDI_修正pop_density_添加金融发展水平.xlsx`
- **二产占比来源**: 从 `人均GDP+人口集聚程度+第二产业占GDP比重+外商投资水平+人均道路面积\总数据集_2007-2023_含第二产业比重.xlsx` 提取
- **最终数据集**: `总数据集_2007-2023_含二产占比_含金融发展水平.xlsx`

### 数据合并
- 成功将 `secondary_share` 变量合并到包含 `financial_development` 的数据集中
- 缺失值数量: 12个观测 (0.25%)
- 合并后样本量: 4,819个观测 × 29个变量

## 平衡性检验结果

### 标准化偏差对比

| 变量 | 匹配前偏差% | 匹配后偏差% | p值 (匹配后) | 偏差减少% | 平衡性 |
|------|-------------|-------------|--------------|-----------|--------|
| ln_pgdp | 44.88 | 0.79 | 0.8296 | 98.2% | ✓ 是 |
| ln_pop_density | 50.58 | -0.67 | 0.8552 | 101.3% | ✓ 是 |
| secondary_share | -8.95 | -12.96 | 0.0004 | 44.8% | ✗ 否 |
| ln_road_area | -24.07 | 0.71 | 0.8476 | -102.9% | ✓ 是 |
| financial_development | 45.96 | -1.54 | 0.6753 | 103.4% | ✓ 是 |

### 平衡性评估

- **满足平衡性标准 (|bias| < 10%)**: 4/5 变量 (80%)
- **优秀**: ln_pgdp, ln_pop_density, ln_road_area, financial_development
- **需关注**: secondary_share (匹配后偏差 -12.96%, p=0.0004)

### 关键发现

1. **大部分变量平衡性优秀**: 4个变量在匹配后偏差均小于1%, p值均>0.05,说明处理组和对照组在这些变量上无显著差异。

2. **secondary_share 未达到平衡性标准**:
   - 匹配前偏差: -8.95% (已在10%阈值内)
   - 匹配后偏差: -12.96% (超出阈值)
   - p值: 0.0004 (高度显著)
   - 偏差反而增加 44.8% (负向)

3. **可能原因**:
   - secondary_share 可能是处理变量 (低碳试点政策) 的重要决定因素
   - 处理组城市本身产业结构与对照组存在系统性差异
   - 需要在DID回归中重点控制此变量

## 结论与建议

### 优点
1. **高匹配率**: 94.2%的平均匹配率,样本损失少
2. **大部分变量平衡性优秀**: 4/5变量匹配后偏差<1%
3. **包含金融发展水平**: 新增了重要的控制变量
4. **二产占比作为控制变量**: 虽然未达到平衡,但可以捕捉产业结构效应

### 需要注意
1. **secondary_share未平衡**: 在DID回归中需要特别关注此变量
2. **ln_road_area缺失率较高**: 24.15%的缺失率导致样本损失
3. **建议**: 在DID回归中,将secondary_share作为核心控制变量,重点关注其系数和显著性
