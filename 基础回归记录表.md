# 基础回归记录表

**创建日期：** 2025年1月6日
**分析内容：** DID基准回归（双向固定效应模型）

---

## 一、回归目的

评估中国低碳城市试点政策对碳排放强度的影响，使用多期双重差分（DID）方法，控制城市和年份固定效应。

---

## 二、数据准备

### 2.1 面板数据结构

- **截面维度：** 城市名称（city_name）
- **时间维度：** 年份（year）
- **数据类型：** 平衡面板
- **城市数量：** 216个地级市
- **时间跨度：** 2007-2023年（17年）
- **总观测数：** 3,672个

### 2.2 变量定义

#### 被解释变量
- **ln_carbon_intensity：** 碳排放强度对数（吨/亿元，2000年基期）

#### 核心解释变量
- **did：** DID政策变量（Treat × Post）
  - Treat = 1：试点城市
  - Post = 1：政策实施后（年份 ≥ 试点年份）

#### 控制变量（5个）
1. **ln_pgdp：** 人均GDP对数（经济水平）
2. **ln_pop_density：** 人口密度对数（人口集聚）
3. **tertiary_share：** 第三产业比重（产业结构）
4. **ln_fdi：** FDI对数（外资水平）
5. **ln_road_area：** 人均道路面积对数（基础设施）

---

## 三、回归模型设定

### 3.1 模型形式

**双向固定效应模型（TWFE）：**

```
ln(CEI_it) = α₀ + β₁·did_it + β₂·ln_pgdp_it + β₃·ln_pop_density_it
             + β₄·tertiary_share_it + β₅·ln_fdi_it + β₆·ln_road_area_it
             + μᵢ + νₜ + ε_it
```

其中：
- CEI_it：城市i在t年的碳排放强度（对数）
- did_it：DID政策变量
- μᵢ：城市固定效应（控制城市层面不随时间变化的因素）
- νₜ：年份固定效应（控制时间层面所有城市共同面临的冲击）
- ε_it：误差项

### 3.2 标准误处理

**聚类稳健标准误：**
- 聚类层面：城市（city_name）
- 聚类数量：216个城市
- 原因：同一城市在不同年份的碳排放存在序列相关性，聚类可避免标准误低估

### 4.2 关键发现

#### 模型(1) - 无控制变量

**DID系数：**
- 系数值：0.0186
- 标准误：0.0107
- t统计量：1.7356
- p值：0.0827
- **显著性：在10%水平上显著**

**经济含义：**
- 低碳试点政策使碳排放强度增加1.88%
- 模型解释力：R² = 0.9690（城市和年份固定效应已解释大部分变异）

#### 模型(2) - 含控制变量

**DID系数：**
- 系数值：0.0204
- 聚类稳健标准误：0.0276
- t统计量：0.7400
- p值：0.4593
- **显著性：不显著**

**经济含义：**
- 控制经济水平、人口、产业、外资和基础设施后，政策效应为正但不显著
- 碳排放强度增加2.06%，但统计上不显著区分于0

**控制变量效应：**
1. **ln_pgdp（-0.5364***）：** 人均GDP每增加1%，碳排放强度降低0.54%（高度显著）
2. **ln_pop_density（-0.3345***）：** 人口密度每增加1%，碳排放强度降低0.33%（高度显著）
3. **tertiary_share（-0.4610***）：** 第三产业比重每增加1单位，碳排放强度降低0.46（高度显著）
4. **ln_fdi（-0.0041）：** FDI影响不显著
5. **ln_road_area（0.0607***）：** 人均道路面积每增加1%，碳排放强度增加0.06%（显著）

### 4.3 模型对比

**解释力提升：**
- R²从0.9690提升至0.9717（+0.27%）
- 调整R²从0.9669提升至0.9698（+0.29%）

**DID系数变化：**
- 从0.0186变为0.0204，变化较小
- 标准误增大（从0.0107到0.0276），显著性下降

## 五、结果解释与讨论

### 5.1 政策效应不显著的可能原因

1. **样本异质性：**
   - 三批试点城市（2010、2012、2017）可能存在异质性效应
   - 不同批次、不同区域的试点效果差异较大

2. **时间滞后效应：**
   - 低碳政策效果可能需要更长时间才能显现
   - 2017年第三批试点距离2023年仅6年，效果可能尚未充分体现

3. **政策实施力度：**
   - 各城市政策执行力度不一
   - 部分城市可能仅停留在规划层面，实际减排措施有限

4. **控制变量吸收效应：**
   - 控制变量（尤其是经济发展和产业结构）可能吸收了部分政策效应
   - 低碳试点政策通过促进产业结构升级间接减排，直接效应被控制变量吸收

### 5.2 控制变量的经济含义

**负向效应变量（降低碳排放）：**
1. **人均GDP：** 支持环境库兹涅茨曲线，经济发展到一定阶段后技术进步推动减排
2. **人口密度：** 集聚效应带来基础设施共享和能源利用效率提升
3. **第三产业比重：** 产业结构从高碳工业向低碳服务业转型

**正向效应变量（增加碳排放）：**
1. **人均道路面积：** 交通基础设施扩张带来机动车使用增加，推高碳排放

### 5.3 下一步分析建议

1. **异质性分析：** 分批次、分区域、分城市规模检验政策效应
2. **平行趋势检验：** 验证DID模型的平行趋势假设
3. **动态效应分析：** 事件研究（Event Study）考察政策动态效应
4. **机制检验：** 检验政策通过产业结构、技术创新等渠道的减排机制
5. **稳健性检验：** PSM-DID、安慰剂检验、排除干扰政策

## 六、回归操作记录

### 6.1 使用的软件和包

- **Python版本：** 3.8.1
- **核心包：**
  - pandas 2.0.3（数据处理）
  - numpy 1.24.4（矩阵运算）
  - scipy（统计推断）
- **回归方法：** 最小二乘虚拟变量法（LSDV）手动实现

### 6.2 估计方法

**OLS估计公式：**
```
β = (X'X)^(-1)X'y
```

**聚类稳健标准误（城市层面）：**
```
VCE_cluster = (X'X)^(-1) [Σ(X_j'·e_j·e_j'·X_j)] (X'X)^(-1)
```
其中j表示城市聚类

### 6.3 固定效应处理

**虚拟变量法：**
- 城市固定效应：215个城市虚拟变量（216个城市，去掉1个基准组）
- 年份固定效应：16个年份虚拟变量（17个年份，去掉1个基准年）
- 总固定效应数：231个

**变量总数：**
- 模型(1)：232个（1个DID + 231个固定效应）
- 模型(2)：237个（1个DID + 5个控制变量 + 231个固定效应）

### 6.4 缺失值处理

- **ln_fdi变量：** 204个观测缺失（5.56%）
- **处理方式：** 使用均值填充（不影响DID系数估计）

## 八、结论

### 8.1 主要结论

1. **基准回归显示：** 低碳试点政策对碳排放强度的直接影响**统计上不显著**
   - DID系数：0.0204（p=0.4593）
   - 经济含义：政策使碳排放强度增加2.06%（但不显著）

2. **控制变量效应显著：**
   - 经济发展、人口集聚、产业结构升级均显著降低碳排放
   - 基础设施扩张增加碳排放

3. **模型解释力强：** R² > 0.97，固定效应和控制变量解释了大部分变异

### 8.2 研究启示

**理论启示：**
- 低碳政策的直接效应可能有限，需更多关注间接机制
- 政策可能通过产业结构调整、技术创新等渠道间接发挥作用

**政策启示：**
- 需加强政策执行力度和配套措施
- 应因地制宜，针对不同城市特点制定差异化政策
- 需长期跟踪评估，政策效果存在时间滞后

### 8.3 后续研究计划

1. **异质性分析：** 分批次、分区域、分规模
2. **平行趋势检验：** 验证DID假设
3. **动态效应：** 事件研究
4. **机制检验：** 产业结构、技术创新
5. **稳健性检验：** PSM-DID、安慰剂检验
