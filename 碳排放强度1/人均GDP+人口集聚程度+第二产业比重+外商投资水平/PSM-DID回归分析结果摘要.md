# PSM-DID回归分析结果摘要

## 控制变量组合
- **ln_pgdp**: 人均GDP (对数)
- **ln_pop_density**: 人口密度 (对数)
- **tertiary_share**: 第三产业占GDP比重 (水平值)
- **ln_fdi**: 外商直接投资 (对数)

## 样本信息
- **原始样本**: 4,819 观测 × 285 城市
- **PSM匹配后**: 3,034 观测 × 201 城市
- **样本损失**: 1,785 观测 (37.0%)
- **匹配率**: 95.7% (各年份)
- **时间范围**: 2007-2023 (17年)

## PSM匹配质量

### 倾向得分模型
- **方法**: 逐年Logit回归
- **McFadden R2**: 0.056-0.086 (平均约0.067)
- **处理组平均PS**: 0.492-0.508
- **对照组平均PS**: 0.408-0.421
- **PS标准差**: 0.126-0.147

### 平衡性检验

| 变量 | 匹配前偏差 | 匹配后偏差 | 偏差减少 | 匹配后平衡 |
|------|-----------|-----------|---------|-----------|
| ln_pgdp | 40.83%*** | -6.63% | 116.2% | ✓ |
| ln_pop_density | 39.58%*** | -2.44% | 106.2% | ✓ |
| tertiary_share | 45.62%*** | 24.85%*** | 45.5% | ✗ |
| ln_fdi | 34.78%*** | -0.52% | 101.5% | ✓ |

**平衡性标准**: |bias| < 10%
**满足标准**: 3/4 变量

**注**: `tertiary_share` 匹配后仍有较大偏差 (24.85%)，说明产业结构在处理组和对照组间存在系统性差异，PSM未能完全消除这一差异。

## PSM-DID回归结果

### 模型设定
```
ln(CEI) = α + β·DID + γ₁·ln(pgdp) + γ₂·ln(pop_density)
          + γ₃·tertiary_share + γ₄·ln(fdi)
          + μᵢ (城市FE) + νₜ (年份FE) + ε
```

### 回归统计量
- **R2**: 0.9722
- **调整R2**: 0.9700
- **样本量**: 3,034
- **城市数**: 201
- **聚类数**: 201 (城市层面)

### 核心结果

| 变量 | 系数 | 聚类稳健标准误 | t统计量 | p值 | 显著性 |
|------|------|--------------|--------|-----|--------|
| **DID (政策效应)** | 0.0376 | 0.0299 | 1.2552 | 0.2109 |  |
| ln_pgdp | -0.4286 | 0.1439 | -2.9789 | 0.0033 | *** |
| ln_pop_density | -0.4011 | 0.1579 | -2.5407 | 0.0118 | ** |
| tertiary_share | -0.1920 | 0.2093 | -0.9175 | 0.3600 |  |
| ln_fdi | 0.0030 | 0.0101 | 0.2968 | 0.7669 |  |

注: *** p<0.01, ** p<0.05, * p<0.1

### 政策效应解读
- **DID系数**: 0.0376
- **经济含义**: 低碳试点政策使碳排放强度增加了 3.76%
- **统计显著性**: p = 0.2109 (不显著)
- **结论**: 低碳试点政策对碳排放强度没有显著的因果效应

### 控制变量系数解读
1. **ln_pgdp (-0.4286***)**: 人均GDP每增加1%，碳排放强度降低0.43%，符合环境库兹涅茨曲线
2. **ln_pop_density (-0.4011**)**: 人口密度每增加1%，碳排放强度降低0.40%，表明集聚效应有利于减排
3. **tertiary_share (-0.1920)**: 第三产业占比每增加1个百分点，碳排放强度降低0.19%，但不显著
4. **ln_fdi (0.0030)**: FDI每增加1%，碳排放强度增加0.003%，影响微弱且不显著

## 研究结论

### 主要发现
1. **政策效应**: 低碳试点政策对碳排放强度无显著的因果效应 (p=0.211)
2. **控制变量**: 经济发展水平和人口集聚程度显著影响碳排放强度
3. **模型拟合**: R2=0.9722，模型解释力强
4. **平衡性**: 3/4变量满足平衡性标准，`tertiary_share`存在残余偏差

### 研究优势
1. **双重稳健估计**: PSM减少选择偏差 + 回归控制剩余混杂因素
2. **固定效应**: 城市和年份双向FE控制不可观测因素
3. **聚类稳健标准误**: 允许城市内序列相关
4. **逐年匹配**: 允许倾向得分随时间变化

### 局限性
1. **样本损失**: 37%观测因缺失值被删除，可能影响代表性
2. **tertiary_share偏差**: 匹配后仍有24.85%偏差，产业结构系统性差异未完全消除
3. **线性假设**: 假设控制变量与ln(CEI)呈线性关系
4. **异质性**: 未考虑不同批次、不同城市的政策效应差异

## 后续分析建议

1. **机制检验**: 考察产业结构升级是否为政策作用渠道
2. **异质性分析**: 分批次、分地区、分城市规模分析
3. **动态效应**: 事件研究法检验政策逐年效应
4. **安慰剂检验**: 随机分配政策时间/城市
5. **排除其他政策**: 控制同时期的其他环境政策
