# 低碳城市建设对产业结构升级的影响研究

## 项目简介

本毕业论文项目研究低碳城市建设对产业结构升级的影响，基于中国家庭金融调查（CFPS）数据和全国地级市经济数据，运用双重差分法（DID）等计量经济学方法进行实证分析。

**作者：** 王晓阳
**学号：** 202211109080
**学院：** 湾区国际商学院

---

## 研究背景

在全球气候变化和可持续发展目标的背景下，中国自2010年起陆续开展了三批低碳城市试点政策。本研究旨在探讨：
- 低碳城市试点政策是否有效推动了产业结构升级
- 政策效果在不同城市间的异质性
- 政策实施的作用机制和传导路径

## 数据说明

### 1. 微观调查数据

**来源：** 中国家庭金融调查（China Family Panel Studies, CFPS）

**涵盖年份：**
- 2010年（基线调查）
- 2012年（第1轮追踪调查）
- 2014年（第2轮追踪调查）
- 2016年（第3轮追踪调查）
- 2018年（第4轮追踪调查）
- 2020年（第5轮追踪调查）
- 2022年（第6轮追踪调查）

**数据内容：**
- 家庭经济状况
- 个人层面的社会经济特征
- 儿童代理调查信息
- 家庭经济行为数据

### 2. 宏观经济数据

**时间范围：** 2000-2023年

**涵盖范围：** 全国330个地级市

**主要指标：**
- GDP及三次产业增加值
- 产业结构水平（第三产业增加值/GDP）
- 产业整体升级指数
- 产业结构高级化指数（第三产业增加值/第二产业增加值）

**计算公式：**
- 产业结构水平 = 第三产业增加值 / 地区生产总值
- 产业整体升级 = Σ(各产业占GDP比重 × 产业层级)
- 产业结构高级化 = 第三产业增加值 / 第二产业增加值

**数据来源：** 各省市统计年鉴、国民经济和社会发展统计公报

## 研究方法

### 计量经济学方法

1. **双重差分法（DID）**
   - 将低碳城市试点作为准自然实验
   - 比较试点城市与非试点城市的产业结构变化差异

2. **倾向得分匹配（PSM）**
   - 解决样本选择偏差问题
   - 提高政策效应评估的准确性

3. **中介效应模型**
   - 检验政策传导机制
   - 分析技术创新、能源消费结构等中介变量
