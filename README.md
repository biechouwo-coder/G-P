# 低碳城市建设对碳排放强度的影响研究

**纳入人口密度数据的低碳试点城市政策影响碳排放强度计量实证研究**

**作者：** 王晓阳
**学号：** 202211109080
**学院：** 湾区国际商学院
**指导教师：**
**研究时间：** 2024-2025年

## 📋 研究摘要

在全球气候变化与中国"双碳"目标（2030年碳达峰、2060年碳中和）的宏观背景下，城市作为能源消耗和温室气体排放的主要来源（贡献全球约70%能源消耗和75%温室气体排放），其低碳转型具有决定性意义。自2010年起，中国分三批启动了低碳省区和低碳城市试点工作。本研究旨在评估这些低碳试点政策对城市**碳排放强度**的因果效应，**核心创新在于将高精度人口密度数据纳入多期双重差分（DID）模型**，以控制城市集聚效应对碳排放的影响，从而更纯净地识别政策净效应。

**关键创新：**
- ✅ 传统研究仅控制"总人口"，本研究引入"人口密度"以反映空间分布形态
- ✅ 区分"集聚正效应"（基础设施共享→降低人均碳排放）与"拥塞负效应"（过度拥挤→推高碳排放）
- ✅ 使用298个城市、27年（1998-2024）的高精度无缺失人口密度数据
- ✅ 采用多期DID（Time-Varying DID）+ 双向固定效应模型 + 倾向得分匹配（PSM-DID）
- ✅ 融合外商投资数据（FDI），控制开放经济因素对碳排放的影响

**主要发现：**
- ⚠️ **低碳试点政策对碳排放强度无显著减排效果**
- ⚠️ **政策效应系数：+2.75%至+4.27%（均不显著，p>0.1）**
- ⚠️ **长期效应（≥5年）：+4.0%至+5.96%（边际不显著，p>0.05）**
- ✅ **平行趋势检验通过**（预处理期p=0.263-0.408，支持DID有效性）
- ✅ 人均GDP与碳排放强度呈显著负相关（环境库兹涅茨曲线验证）
- ✅ 产业结构高级化（三产/二产）显著降低碳排放强度

## 🎯 研究目标

### 主要研究问题

1. **政策效应评估**
   - 低碳试点政策是否有效降低了城市碳排放强度？
   - 政策效应有多大？（净效应多少？）

2. **机制识别**
   - 政策通过何种渠道影响碳排放？
   - 产业结构调整？技术创新？能源结构优化？

3. **集聚效应控制**
   - 在控制人口密度后，政策净效应有多大？
   - 人口密度呈现集聚效应还是拥塞效应？

4. **异质性分析**
   - 政策效果在不同城市（发达vs.发展中）有何差异？
   - 不同批次试点（2010 vs. 2012 vs. 2017）效果是否不同？
   - 区域异质性（东部vs.中部vs.西部）？

### 核心创新点

| 创新维度 | 传统研究 | 本研究 |
|---------|---------|--------|
| **人口变量** | 仅使用总人口 | 引入高精度人口密度（298城市×27年，无缺失） |
| **模型设定** | 传统单时点DID | 多期DID + 双向固定效应 |
| **理论框架** | 简单STIRPAT | 扩展STIRPAT（人口规模+空间分布形态） |
| **稳健性** | 基础检验 | 平行趋势+PSM-DID+安慰剂+排除干扰政策 |

## 🔬 研究方法

### 理论框架：扩展STIRPAT模型

本研究的变量选取基于环境经济学经典模型 **STIRPAT** (Stochastic Impacts by Regression on Population, Affluence, and Technology)：

```
I = P × A × T
```

**传统STIRPAT的局限：**
- 仅关注**人口规模（P）**，忽略空间分布
- 无法区分"人多"与"拥挤"的不同影响

**本研究的扩展：**
引入**人口密度**以反映空间分布形态

**理论逻辑：**
- **集聚正效应：** 高密度 → 基础设施共享 → 公共交通规模经济 → **人均碳排放下降**
- **拥塞负效应：** 过度拥挤 → 交通拥堵 → 能源需求激增 → **人均碳排放上升**

通过控制人口密度，本研究试图剥离城市化形态对碳排放的影响，更纯净地识别政策制度性减排效应。

### 低碳试点政策演进

中国低碳试点政策分三批实施，构成理想的"准自然实验"：

| 批次 | 年份 | 范围 | 重点 |
|------|------|------|------|
| **第一批** | 2010年 | 5省8市 | 摸清家底，编制温室气体排放清单 |
| **第二批** | 2012年 | 29个省市 | 建立碳排放管理体系，探索碳交易 |
| **第三批** | 2017年 | 45个城市 | 聚焦碳达峰，强调技术创新与产业调整 |

这种渐进式政策推进为多期DID模型提供了理想的识别场景。

### 基准回归模型：多期DID

鉴于政策分三批实施，采用多期双重差分模型：

**主规格（产业结构高级化）：**
```
ln(CEI) = α + β·DID + γ₁·ln(pgdp) + γ₂·ln(pop_density)
          + γ₃·industrial_advanced + γ₄·fdi_openness
          + γ₅·ln(road_area) + μᵢ + νₜ + ε
```

**替代规格（第三产业占比）：**
```
ln(CEI) = α + β·DID + γ₁·ln(pgdp) + γ₂·ln(pop_density)
          + γ₃·tertiary_share + γ₄·tertiary_share²
          + γ₅·ln(fdi) + γ₆·ln(road_area) + μᵢ + νₜ + ε
```

**变量说明：**
- `ln(CEI_it)`：碳排放强度对数（因变量，已对数化）
- `DID_it = Treat_i × Post_it`：核心解释变量（双重差分项）
  - `Treat_i`：是否为试点城市（0/1）
  - `Post_it`：政策实施后（2010/2012/2017年起为1）
- `ln(pop_density_it)`：**人口密度对数（核心控制变量）**
- `industrial_advanced`：产业结构高级化（三产/二产，反映产业升级）
- `fdi_openness`：外商投资开放度（FDI/名义GDP，控制开放经济效应）
- `Control_kit`：其他控制变量（人均GDP、人均道路面积等）
- `μᵢ`：城市固定效应（吸收地理、气候、资源禀赋等）
- `νₜ`：年份固定效应（吸收宏观冲击、国家政策变化）
- `ε_it`：随机扰动项

**系数含义：**
- `β`：政策净效应（若<0且显著，表明政策有效降低碳强度）
- `γ₂`：人口密度弹性（若<0支持集聚效应，若>0支持拥塞效应）

**估计方法：**
- ✅ 最小二乘虚拟变量（LSDV）法估计双向固定效应
- ✅ 标准误聚类到城市层面（允许序列相关）
- ✅ 所有连续变量取对数并缩尾处理（1%/99%分位数）

### 稳健性检验策略

1. **平行趋势检验** ⭐ ✅ 已完成
   - 事件研究法（Event Study）
   - 检验试点前处理组与控制组趋势是否一致
   - **结果：** 预处理期系数不显著（p=0.263-0.408），支持平行趋势假设

2. **PSM-DID（倾向得分匹配双重差分）** ✅ 已完成
   - 解决试点城市选择非随机问题
   - 以人口密度、人均GDP、产业结构为协变量
   - 1:1近邻匹配（有放回），在匹配样本上重新回归
   - **结果：** 匹配后回归系数仍不显著，结论稳健

3. **排除干扰政策** ⏳ 待进行
   - 控制"智慧城市"、"创新型城市"、"海绵城市"等试点
   - 验证低碳试点效应的独立性

4. **替换被解释变量** ⏳ 待进行
   - 人均碳排放量（CO₂/Population）
   - 碳排放总量对数（ln(CO₂)）
   - 排除仅由GDP增长带来的"伪减排"

5. **安慰剂检验** ⏳ 待进行
   - 蒙特卡洛模拟（500-1000次）
   - 随机分配试点城市和实施时间
   - 验证真实系数显著异于随机分布

6. **工具变量法（IV）** ⏳ 待进行
   - 解决潜在双向因果关系
   - 地理工具变量（空气流通系数、河流密度）
   - Bartik工具变量（邻近城市政策情况）
   - 2SLS估计

## 🔍 主要研究发现总结

### 政策效应评估

| 模型规格 | 样本量 | DID系数 | p值 | 显著性 | 平行趋势检验 |
|---------|--------|---------|-----|--------|-------------|
| **主规格**（产业高级化+fdi_openness） | 2,846×199城市 | +0.0275 | 0.329 | 不显著 | ✅ 通过（p=0.263） |
| **替代规格**（第三产业占比） | 2,990×200城市 | +0.0427 | 0.105 | 不显著 | ✅ 通过（p=0.408） |

**结论：** 低碳试点政策对碳排放强度无显著减排效果，两种规格结论一致，结果稳健。

### 动态效应（事件研究）

**主规格结果：**
- 短期效应（t=0）：+2.2%（p=0.438，不显著）
- 中期效应（t=1至+4）：-3.4%至+8.9%（均不显著）
- 长期效应（t≥+5）：+4.0%（p=0.262，不显著）

**替代规格结果：**
- 长期效应（t≥+5）：+5.96%（p=0.081，边际不显著）

**解释：** 政策实施5年后，排放强度反而可能上升约4-6%（边际不显著），可能暗示存在"污染避难所效应"或"基础设施扩张效应"。

### 控制变量效应

| 变量 | 预期符号 | 实际符号 | 显著性 | 经济含义 |

| ln(人口密度) | ? | + | ✅ *** | 拥塞效应：高密度推高碳强度 |
| 产业高级化（三产/二产） | - | - | ✅ *** | 结构优化：服务业替代工业减排 |
| 外商投资开放度 | ? | + | 不显著 | 开放经济的碳排放效应不明确 |

**政策建议：**
1. 低碳试点政策需要重新设计，加强执行力度
2. 产业结构升级（提高三产占比）是有效减排途径
3. 外商投资政策需与环境规制协调，防范"污染避难所"
